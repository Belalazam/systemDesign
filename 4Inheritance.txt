inheritance -> it allows one class to inherit properties of another class


why inheritance->
code reusability, code can be extended easily, manitanice easy etc


let see some syntax->

class parent{
    public:
    void testParent()
    {
        cout<<"hi, i am parent";
    }
};

class child: public parent{
   public:
    void testChild()
    {
        cout<<"hi, i am child";
    }
};
